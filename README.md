## Overview
Relational database project analyzing digital marketing campaign performance using SQL. Calculates revenue, ROI, and ROI per day across campaigns, ad types, and platforms. Designed for integration with a PHP or API driven backend.

## Business Use Case
Helps teams identify which campaigns and ad strategies generate real value, not just engagement.

## Data Model
users: user info
campaigns: campaign metadata and budget
ads_new: ad level targeting and platform data
events: impressions, clicks, purchases

## Key Metrics
Revenue from purchase events
ROI and ROI per day
Performance by ad type and platform

## Skills Demonstrated
SQL joins, aggregation, and CASE logic
Relational database design
Business metric calculation
Data integrity and edge case handling
Git based project documentation

## Tech Stack
MySQL | SQL | PHP (ready) | Git
